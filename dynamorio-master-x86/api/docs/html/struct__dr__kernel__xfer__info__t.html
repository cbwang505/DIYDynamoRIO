◆ source_mcontext

const dr_mcontext_t* _dr_kernel_xfer_info_t::source_mcontext

The source machine context which is about to be changed. This may be NULL if it is unknown, which is the case for DR_XFER_CALLBACK_DISPATCHER and DR_XFER_RSEQ_ABORT (where the PC is not known but the rest of the state matches the current state).
